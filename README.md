This repository contains the high-fidelity CFD dataset of variable-geometry hypersonic inlets, developed to support research on reduced-order modeling (ROM), aerodynamic design optimization, and data-driven flow field prediction for hypersonic air-breathing propulsion systems. This dataset consists of 220 samples of hypersonic inlet flow fields with varying geometric configurations. Each sample includes full-field flow variables from validated CFD simulations, suitable for developing and benchmarking data-driven models for hypersonic flow prediction, especially for scenarios with strong shock discontinuities and complex wave structures. para_220.csv：Geometric design parameter table for all 220 inlet samples, with each row corresponding to the design parameters of one sample, matched one-to-one with the flow field data. all_grid_coords_ma.zip：Full flow field dataset, including grid coordinates and snapshot matrices of all primary flow variables for all samples, stored in numpy format for Python compatibility. flow_field_snapshot_ma.zip：Mach number field-specific dataset, containing grid coordinates and Mach number field matrices for all samples, optimized for lightweight applications such as flow feature extraction and reduced-order modeling.
